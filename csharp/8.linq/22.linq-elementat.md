
# ElementAt
Argument	Type
Source sequence	IEnumerable<TSource>
Index of element to return	int
ElementAt picks the nth element from the sequence:
```c#
int[] numbers  = { 1, 2, 3, 4, 5 };
int third      = numbers.ElementAt (2);            // 3
int tenthError = numbers.ElementAt (9);            // Exception
int tenth      = numbers.ElementAtOrDefault (9);   // 0
```
Enumerable.ElementAt is written such that if the input sequence happens to implement IList<T>, it calls IList<T>â€™s indexer. Otherwise, it enumerates n times and then returns the next element. ElementAt is not supported in EF Core.
